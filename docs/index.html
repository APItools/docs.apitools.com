Using APIs has never been easier! Stay in absolute control of API traffic by proxying your calls to any API through APItools. In the cloud or on-premise. And we're open-source.

Use APItools to easily get started with any API, save time when troubleshooting, monitor end users usage of an API, modify HTTP(s) requests on the fly, amongst many other things.

Step 1 – Set Up An API Service

Create a traffic monitor to be able to set up your first API service. You can add any internal or external API service that you're using.

Step 2 – Push Through Some API Traffic

Go to the 'Integration' tab to learn how to link APItools to your app. Then run your app, make a cURL request or click on an example HTTP(s) request to push some traffic through. Go to the 'Traces' tab and verify you hit the API.
